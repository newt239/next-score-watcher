# 共通操作方法

このドキュメントでは、Score Watcherのすべてのゲーム形式で共通する操作方法について説明します。

## UI表示・操作方法

### スコアボード表示

- プレイヤーカードに各ゲーム形式に応じたスコア表示ボタンが配置される
- 勝利状態のプレイヤーは赤色の背景で強調表示
- 失格状態のプレイヤーは青色の背景で強調表示

### 操作方法

#### マウス操作

- **スコア表示ボタンをクリック**: 正解として記録され、スコアが変動
- **各ゲーム形式固有のボタン**: 形式によって異なる操作ボタンが表示される場合があります

#### キーボードショートカット

- **数字キー（1-9, 0）**: 対応するプレイヤーを正解として記録
- **Shift + 数字キー**: 対応するプレイヤーを誤答として記録
- **カンマキー (,) または Ctrl+Z**: 最後のアクションを元に戻す
- **ピリオドキー (.)**: スルー（問題をスキップ）

#### 編集モード

- 編集モードを有効にすると、スコア表示部分が入力フィールドになり、直接スコアを編集可能
- 編集モードでの直接スコア変更は、ログには記録されないため、元に戻す機能では復元されません

### 共通操作ボタン

- **スルーボタン**: 問題をスキップ
- **一つ戻すボタン**: 最後のアクションを取り消し
- **スコア手動更新ボタン**: 編集モードの切り替え
- **フルスクリーンボタン**: 全画面表示切り替え
- **表示設定ボタン**: 表示オプション変更
- **ゲーム設定ボタン**: ゲーム設定画面へ移動

## 注意事項

- 勝利・失格の判定は即座に行われ、状態変更後も操作は可能ですが、状態表示は維持されます
- キーボードショートカットは、対応するプレイヤー数まで有効です（最大10人まで）
- ゲーム進行中の設定変更は、現在のゲーム状態に影響を与える場合があります
